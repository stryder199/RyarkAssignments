Generic Verification Process for Question Templates

This document specifies the aspects of the testing and review process common to all question templates in the CQG system.

The verification tasks specific to question type QT are found in
testing_review/QT/verification_template_verification_plan.html.

Review Process

Safe
Show that each generated question is safe: regardless of the answer, the check answer computation will terminate normally.
Solvable
For each generated question, there is at least one answer which will be marked "correct".
Accurately marked
Each answer, the check answer calculation is accurate.

Test Process

Template verifier
Execute the template verifier against the template.
