Question template format: multiple_choice

This document explains the syntax and semantics of the multiple_choice question template.

The required variables and the restrictions on their values are specified by the Template Verification Conditions.

0-relative index indicates correct answer Questions and answers may contain limited html.

The variable group_list must contain a list of groups, where each group has the form:

[group_name, question_vector_0, question_vector_1, ...]

and each question vector has the form: 

[question_text, answers,correct_answer]
One question is generated for each question vector. The question generated from question_vector_i is stored in a directory with name constructed by appending i to group_name.

answers is a list of possible answers, and correct_answer is the index of the correct answer (if only one answer is correct) or a list of indeces of correct answers (if multiple are correct).
